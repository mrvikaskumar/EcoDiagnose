# 🌱 EcoDiagnose

**Pioneering smart e-waste management through AI triage and decentralized recycling networks.**

EcoDiagnose is a comprehensive full-stack solution built to tackle the growing e-waste crisis. By bridging the gap between consumers and certified recycling partners, the platform facilitates the responsible disposal and component harvesting of old electronics. 

## 🚀 Key Features

* **AI-Powered Triage:** Integrates Python-based AI to scan and assess the condition of uploaded electronics, calculating a precise damage score (0-100) to determine recycling viability.
* **Decentralized Partner Network:** A dedicated portal for registered recycling distributors to claim local e-waste pickup requests.
* **Role-Based Access Control (RBAC):** Secure, isolated environments for Regular Users, Recycling Partners, and System Administrators.
* **Automated Admin Controls:** A centralized command dashboard featuring a 5-strike automated warning and blocking system for partner moderation, complete with direct email integration.
* **High-Security Data Validation:** Implements strict Regex validations for Indian GSTIN formats, 10-digit mobile numbers, and pincodes to ensure data integrity.
* **Responsive UX/UI:** A fully responsive, mobile-first design built with React and Tailwind CSS.

## 💻 Tech Stack

* **Frontend:** React.js, Tailwind CSS, React Router
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas, Mongoose
* **AI Integration:** Python
* **Authentication & Security:** JWT (JSON Web Tokens), Bcrypt, HTML5 Native Validation
* **Mailing:** Nodemailer
